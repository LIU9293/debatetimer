# Javascript Based Single Page Application - Debate Timer

### Head
***

点击[这里](http://timer.party)可以直达

### Game rule
***
1. 正方一辩陈词，3分钟。反方质询1分30秒。 
2. 反方一辩陈词，3分钟。正方质询1分30秒。
3. 双方二辩申论，2分30秒。               
4. 双方二辩对辩，各1分30秒。              
5. 双方二辩盘问，2分30秒。
6. 双方二辩小结，1分30秒。              
7. 自由辩论，各4分钟。  
8. 结辫，3分30秒。   

*奇袭环节：
    质询：2分30秒          
    申论：2分
